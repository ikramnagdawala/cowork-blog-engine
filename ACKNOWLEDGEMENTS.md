# Acknowledgements

The idea for a queue-driven, research-to-draft blog engine that plugs into WordPress was inspired by
Manthan Patel's open-source [`seo-blog-writer`](https://github.com/) (MIT licensed). This project is a
fresh, independent build: the code here was written from scratch and generalised for any niche and any
writer, with a built-in humaniser, a guided setup wizard, and a document output mode for people who
don't use WordPress. The original sparked the approach; the implementation and design are my own.

Thanks, Manthan, for putting the original out there.

— Ikram Nagdawala
